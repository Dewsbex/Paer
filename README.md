# pær

**Many sources. One signal.**

Paer is an RSS/Atom feed reader built for synthesis, not just consumption. It helps knowledge workers see patterns across sources, turn reading into structured output, and stay informed without drowning.

---

## What makes Paer different

**Cross-feed topic clustering** — When multiple sources cover the same story, Paer groups them automatically and synthesises the key points across perspectives.

**Read → think → act** — Highlight text, add notes, and export your annotations as markdown to Obsidian, Notion, or Logseq. Reading flows into action, not a dead end.

**Calm by design** — No unread counts. No badges. No guilt. Time-window filtering shows you the last 15 minutes, 1 hour, 24 hours, or 3 days. When you reach the end, you're done.

**Bring your feeds** — Connect your existing Feedbin, Miniflux, or FreshRSS account. Or start fresh with Paer's built-in backend.

**Accessible** — Atkinson Hyperlegible font (Braille Institute), adjustable text size, WCAG AA contrast, dark mode, keyboard navigation. Accessibility is a feature, not compliance.

---

## Stack

- **Frontend:** SvelteKit + Tailwind CSS
- **Backend:** Python FastAPI + Celery + Redis
- **Database:** PostgreSQL
- **Deployment:** Docker
